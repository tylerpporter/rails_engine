# Rails Engine

API for [Rails Driver](https://github.com/turingschool-examples/rails_driver)

## Deployment

N/A Development only 

## Getting Started

- Fork this repository
- Clone your fork
- Run `rails db:create`
- Run `rails db:migrate`
- Run `rake db:seed:from_csv`

### Prerequisites

- Run `bundle install`


## Running the tests

- Run `bundle exec rspec`
